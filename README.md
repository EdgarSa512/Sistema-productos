# Sistema de Productos 🛒

Proyecto escolar de un supermercado donde varios usuarios ingresan en conjunto para consultar el inventario de la disponibilidad en stock.

## 📋 Descripción

Sistema de gestión de inventario que permite a múltiples usuarios:
- Consultar disponibilidad de productos
- Ver categorías de artículos
- Verificar precios y cantidades en stock

## 🗄️ Base de Datos

La aplicación utiliza MySQL con las siguientes tablas:
- **usuario**: Información de clientes
- **categoria**: Categorías de productos
- **producto**: Inventario de productos

## 📊 Estructura

- Electrónica
- Papelería
- Alimentos
- Herramientas
- Ropa

## 🚀 Instalación

1. Crear la base de datos ejecutando el script SQL
2. Conectar la aplicación a la base de datos
3. Ejecutar consultas sobre el inventario
